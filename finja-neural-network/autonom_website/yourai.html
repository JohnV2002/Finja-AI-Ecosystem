Get Started

Run Your Own

YourAI is fully open-source. Clone, configure, and launch in minutes.

# Clone the ecosystem
git clone https://github.com/AdminV2002/YourAI-AI-Ecosystem

# Navigate to directory
cd YourAI-AI-Ecosystem

# Launch with Docker
docker compose up
