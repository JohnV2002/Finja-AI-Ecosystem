Über Glorpo 🧬

Glorpo ist eine esoterische Programmiersprache, bei der Python-Code wie absoluter Nonsens aussieht — aber nativ funktioniert.

"Glorpo is pain." — Magic The Noah

Die Idee

Was wäre wenn def in Python gloo hieße? Und if → glorb? Und print → glorp?

Glorpo Web bringt das gleiche Chaos ins HTML. Du schreibst normales HTML, PHP glorpifiziert es — und wer F12 drückt, fragt sich ob er halluziniert.

Warum?

  • Weil es geht
  • Weil es lustig ist
  • Weil "Glorpo is pain." und das stimmt
  • Als leichte HTML-Obfuskierung ohne JS-Obfuskierung

Wie funktioniert es?

PHP liest deine normale .html Datei, ersetzt alle Tags durch ihre Glorpo-Äquivalente, und schickt das Ergebnis an den Browser. Eine glorpo.css definiert für jeden Glorpo-Tag das korrekte Rendering.

Du schreibst: <div><h1>Hallo</h1></div>
Browser kriegt: <glorpbox><glorphat>Hallo</glorphat></glorpbox>
